# Introduction
The current project is creating an entire web page from a design TOP has provided.

# Skills
The following skills are needed in order to complete the project:
- Basic HTML elements such as containers, lists, links, buttons.
- Basic CSS such as working with colors, fonts, sizes, margins, paddings, borders.
- Flexbox.

# About
This project is part of [The Odin Project](https://www.theodinproject.com/)'s foundation course. You can find the link to the corresponding lesson [here](https://www.theodinproject.com/lessons/foundations-foundations-landing-page).
